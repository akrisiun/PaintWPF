artesdi description:

Редактор представляет из себя:
1) “рабочую область” -- некоторое количество наложенных друг на друга слоёв;
2) органы управления слоями.

Реализовано:
- Перемешение слоёв в рабочей области друг относительно друга;.
- Добавление нового пустого слоя.
- Удаление слоя.
- Изменение порядка слоёв (влияет на результат наложения).
- Скрытие/показ слоя.
- Рисование только на выделенном слое.
- Дополнительные параметры для редактирования слоя (открываются/закрываются щелчком правой кнопкой мыши на панели слоёв)
- Переименования слоя осуществляется двойным щелчком по названию
- Объединение и сохранение изображения со всех слоёв в формат png
